# QA JavaScript Fundamentals

Este repositório foi criado como parte dos meus estudos para me tornar **QA (Quality Assurance)**.

O objetivo deste projeto é praticar **conceitos básicos de JavaScript e lógica de programação**, habilidades importantes para quem deseja evoluir futuramente para **automação de testes**.

Durante o desenvolvimento das atividades, trabalhei com estruturas fundamentais da linguagem e manipulação de dados simples.

# 📂 Estrutura do projeto

O projeto está dividido em pequenos exercícios para praticar conceitos específicos da linguagem.

### 🔁 Estrutura de Repetição

Percorre uma lista de times exibindo:

- número da execução
- nome do time
- separação entre execuções

Times utilizados:

- Santos
- Flamengo
- Fluminense
- Bahia
- Grêmio

---

### 🧠 Estrutura de Decisão

Percorre uma lista de países verificando se **Brasil** está presente.

Países utilizados:

- Jamaica
- Brasil
- Itália
- Canadá

O código imprime:

- número da execução
- nome do país
- mensagem informando se o país encontrado é Brasil ou não

---

### ⚙️ Funções

Criação de uma função simples que:

- recebe dois números como parâmetro
- realiza a soma
- exibe o resultado no console

Exemplo:

soma(5, 10)

Resultado exibido:

numero1: 5

numero2: 10

soma: 15


---

### 📦 Arrays

Prática de manipulação de arrays utilizando métodos comuns do JavaScript.

Operações utilizadas:

- `push()` → adiciona elemento no final
- `pop()` → remove o último elemento
- `unshift()` → adiciona elemento no início
- `shift()` → remove o primeiro elemento
- `length` → retorna o tamanho do array

Também foi demonstrado como acessar posições específicas do array.
